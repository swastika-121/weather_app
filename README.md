# Weather App (Django)
Developed a modern Django-based Weather App with API integration, real-time weather updates, 5-day forecast, dark mode toggle, and dynamic weather animations. Deployed live using Render, with responsive design using Bootstrap 5.

# Live Demo: https://weather-app-1-1vdf.onrender.com

## Features
- Search weather by city (global support)
- 5-day weather forecast
- Dark mode toggle
- Dynamic weather animations (rain, snow, clouds)
- Secure API handling using environment variables
- Responsive design

## Tech Stack
- Django (Backend)
- Python
- HTML, CSS, JavaScript
- OpenWeatherMap API
- Render (Deployment)
